Java script Promise

A promise is an obj represent completion or failure on an Async operation.

  • Pending:-Inital Stage,Not yet Fulfilled or Rejected
  • Fulfilled/Resolve:-Promise Complted
  • Rejected:-Promise Failed


Promise Components

  • Promise()
  • .then()
  • .catch()
  • .finally()

.then()

  .then() returns A promise
  it will take the following parameters

    1.onResolve:-Callback function for the Success.
    2.onRejected:-Callback function for the failure.

    .then(onResolve,onRejected)
    onResolve:-A function called if the Promise is Fulfilled,This functuion has one argument which is the Fulfilled value of the Promise. 
    onRejected:- A function called if the Promise is rejected,This functuion has one argument which is the Rejected reason of the Promise.
     
//Consuming Code
    promiseObj.then(
        value=>{console.log(value)},
        error=>{console.log(error)}
        )
