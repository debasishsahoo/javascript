The XmlHttpRequestEventTarget.onload is a function called when an XMLHttpRequest transition is completed


The XmlHttpRequestEventTarget.onprogress is a function called periodiclly with information when an XMLHttpRequest before sucess completly


The XmlHttpRequestEventTarget.error is a function called when an XMLHttpRequest transition is fails due to an Error


The XmlHttpRequestEventTarget.onloadstart is a function called when an XMLHttpRequest transition starts transferring Data


The XmlHttpRequestEventTarget.onabort is a function called when an XMLHttpRequest transition is aborted,such as when XMLHttpRequest.abort() is called


The XmlHttpRequestEventTarget.onloadend is a function called when an XMLHttpRequest transition End transferring Data
