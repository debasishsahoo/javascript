readyState

UNSENT UNSENT (numeric value 0) UNSENT open() has not been called yet. The object has been constructed.
opened OPENED (numeric value 1) OPENED send() has been called. The open() method has been successfully invoked. During this state request headers can be set using setRequestHeader() and the fetch can be initiated using the send() method.
headers received HEADERS_RECEIVED (numeric value 2) HEADERS_RECEIVED send() has been called, and headers and status are available. All redirects (if any) have been followed and all headers of a response have been received.
loading LOADING (numeric value 3) LOADING Downloading; responseText holds partial data. The response body is being received.
done DONE (numeric value 4) DONE The operation is complete. The data transfer has been completed or something went wrong during the transfer (e.g., infinite redirects).

An XMLHttpRequest object travels them in the order 0 → 1 → 2 → 3 → … → 3 → 4. State 3 repeats every time a data packet is received over the network.
