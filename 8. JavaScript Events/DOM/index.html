Event Description
click The click event is fired when the user clicks on an element.
mousedown The mousedown event is fired when the user presses the mouse button over an element.
mouseout The mouseout event is fired when the user moves the mouse pointer outside the boundaries of an element.
mouseover The mouseover event is fired when the user moves the mouse pointer over an element.
mouseup The mouseup event is fired when the user releases the mouse button over an element.
mouseenter The mouseenter event is fired when the user moves the mouse pointer from outside an element into the interior of an element.
mouseleave The mouseleave event is fired when the user moves the mouse pointer from the interior of an element to outside the element's boundaries.
mousemove The mousemove event is fired when the user moves the mouse over the document.
dblclick The dblclick event is fired when the user double-clicks on an element.
load The load event is fired when the browser has completed loading all the resources of a document.
unload The unload event is fired when the document has been removed from a window or frame.
reset The reset event is fired when a reset button is clicked.
select The select event is fired when a user selects some text.
resize The resize event is fired when a document view is resized.
scroll The scroll event is fired when a document view is scrolled.
change The change event is fired when a control loses the input focus and its value has been modified.
keydown The keydown event is fired when a user presses a keyboard key.
keypress The keypress event is fired when a user presses a keyboard key.
keyup The keyup event is fired when a user releases a keyboard key.
blur The blur event is fired when an element loses focus.
focus The focus event is fired when an element receives focus.
focusin The focusin event is fired when an element receives focus.
focusout The focusout event is fired when an element loses focus.
select The select event is fired when some text is selected.
abort The abort event is fired when a request has been aborted.
canplay The canplay event is fired when the browser can start playing the media.
canplaythrough The canplaythrough event is fired when the browser can play through the media without stopping for buffering.
durationchange The durationchange event is fired when the duration of the media has been determined.
emptied The emptied event is fired when the media has become empty; for example, this event is sent if the media has already been loaded (or partially loaded), and the load() method is called to reload it.
ended The ended event is fired when the media has reached the end (duration) and is no longer playing.
loadeddata The loadeddata event is fired when the browser has loaded the current frame of the media.
loadedmetadata The loadedmetadata event is fired when the browser has loaded meta data for the media.
loadstart The loadstart event is fired when the browser starts looking for the media.
pause The pause event is fired when the media is paused.
play The play event is fired when the media starts to play.
playing The playing event is fired when the media is playing.
progress The progress event is fired when the browser is downloading the media data.
ratechange The ratechange event is fired when the playing speed of the media has changed.
seeked The seeked event is fired when the user is finished moving/skipping to a new position in the media.
seeking The seeking event is fired when the user starts moving/skipping to a new position in the media.
stalled The stalled event is fired when the browser is trying to get media data, but data is not available.
suspend The suspend event is fired when the browser is intentionally not getting media data.
timeupdate The timeupdate event is fired when the current playback position has changed.
volumechange The volumechange event is fired when the volume has been changed.
waiting The waiting event is fired when the media starts to buffer.
load The load event is fired when the browser has completed loading all the resources of a document.
unload The unload event is fired when the document has been removed from a window or frame.
beforeunload The beforeunload event is fired when the document is about to be unloaded.
blur The blur event is fired when an element loses focus.
change The change event is fired when a control loses the input focus and its value has been modified.
click The click event is fired when the user clicks on an element.
contextmenu The contextmenu event is fired when the user right-clicks on an element.
dblclick The dblclick event is fired when the user double-clicks on an element.
drag The drag event is fired when an element or text selection is being dragged.
dragend The dragend event is fired when a drag operation is being ended (by releasing a mouse button or hitting the escape key).
dragenter The dragenter event is fired when a dragged element or text selection enters a valid drop target.
dragleave The dragleave event is fired when a dragged element or text selection leaves a valid drop target.
dragover The dragover event is fired when an element or text selection is being dragged over a valid drop target (every few hundred milliseconds).
dragstart The dragstart event is fired when the user starts to drag an element or text selection.
drop The drop event is fired when an element or text selection is dropped on a valid drop target.
focus The focus event is fired when an element receives focus.
input The input event is fired when the value of an element changes or the content of an element with the attribute contenteditable is modified.
keydown The keydown event is fired when the user presses a keyboard key.
keypress The keypress event is fired when the user presses a key.
keyup The keyup event is fired when the user releases a key.
mousedown The mousedown event is fired when the user presses a mouse button.
mousemove The mousemove event is fired when the user moves the mouse.
mouseout The mouseout event is fired when the user moves the mouse out of the browser window.
mouseover The mouseover event is fired when the user moves the mouse over a valid drop target.
mouseup The mouseup event is fired when the user releases a mouse button.
mousewheel The mousewheel event is fired when the user rolls the mouse wheel.
scroll The scroll event is fired when the document is scrolled.
select The select event is fired when some text is being selected.
submit The submit event is fired when a form is submitted.
touchcancel The touchcancel event is fired when a touch point is removed from the touch surface.
touchend The touchend event is fired when a touch point is removed from the touch surface.
touchmove The touchmove event is fired when a touch point is moved along the touch surface.
touchstart The touchstart event is fired when a touch point is placed on the touch surface.
wheel The wheel event is fired when the mouse wheel is rotated.
